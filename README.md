# Examen-Parcial-2
Proyecto del examen para el parcial 2 de programacion web

## Tecnologias usadas
**Frontend**
* JavaScript
* HTML
* CSS
**Backend**
* Node.JS
**Librerias usadas para ejecutar el backend**
* Cors
* Express
* Nodemon
* Pdfkit

Recuerda usar 
```markdown
**PowerShell**

npm install express cors nodemon pdfkit
```
si solo quieres descargar la caperta node usa

```markdown
**PowerShell**
npm install 
```
